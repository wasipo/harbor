# Harbor Design System

勤怠管理SaaSのための統一デザインシステム。React Aria Components + Tailwind CSSで構築。

## 🎨 デザイントークン

### カラーパレット

```css
/* Primary Colors - ブランドカラー */
--color-primary-50: #eff6ff;
--color-primary-500: #3b82f6;
--color-primary-600: #2563eb;
--color-primary-700: #1d4ed8;

/* Semantic Colors - 機能的カラー */
--color-success: #10b981;  /* 承認済み */
--color-warning: #f59e0b;  /* 承認待ち */
--color-error: #ef4444;    /* エラー・拒否 */
--color-info: #06b6d4;     /* 情報 */

/* Neutral Colors - グレースケール */
--color-gray-50: #f9fafb;
--color-gray-100: #f3f4f6;
--color-gray-500: #6b7280;
--color-gray-900: #111827;
```

## 🧩 基本コンポーネント

### Button

```tsx
// Primary Button - 主要アクション
<Button variant="primary" size="md">
  勤怠登録
</Button>

// Secondary Button - 副次アクション
<Button variant="secondary" size="md">
  キャンセル
</Button>

// Danger Button - 削除などの危険なアクション
<Button variant="danger" size="md">
  削除
</Button>
```

### Input

```tsx
// Text Input
<Input
  label="従業員名"
  placeholder="山田太郎"
  required
/>

// Date Input
<DatePicker
  label="勤務日"
  required
/>

// Time Input
<TimeField
  label="開始時刻"
  required
/>
```

### Status Badge

```tsx
// 勤怠ステータス
<Badge variant="draft">下書き</Badge>
<Badge variant="submitted">提出済み</Badge>
<Badge variant="approved">承認済み</Badge>
<Badge variant="rejected">却下</Badge>
```
