# Harbor ワイヤーフレーム

勤怠管理システムの画面レイアウト設計。

## 🏠 ダッシュボードレイアウト

ダッシュボードは以下の要素で構成されています：

- **ヘッダー**: ロゴ、ナビゲーション、ユーザーメニュー
- **サイドバー**: メインナビゲーション（モバイルでは非表示）
- **メインコンテンツ**: 
  - 今日の勤怠状況カード
  - 月次サマリー
  - 最近の勤怠履歴
  - お知らせ

## 📱 レスポンシブ対応

- **デスクトップ** (1024px以上): サイドバー + メインコンテンツ
- **タブレット** (768px-1023px): 折りたたみ可能なサイドバー
- **モバイル** (767px以下): ボトムナビゲーション

## 🎨 デザイン原則

1. **明瞭性**: 情報の優先順位を明確に
2. **一貫性**: 同じ操作は同じUIパターンで
3. **アクセシビリティ**: キーボード操作、スクリーンリーダー対応
4. **レスポンシブ**: すべてのデバイスで快適に使用可能

詳細な実装はコンポーネントのStoriesを参照してください。
